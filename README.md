## GitHub rep with two folders 


### structure

```
fintrack/
├── finance-tracker-backend/
└── finance-tracker-frontend/
```

### Steps
keep 2 folders under main folder(fintrack), then
```bash
git init
git add .
git commit -m "Add backend and frontend folders"
git branch -M main
git remote add origin https://github.com/sindhu27b/fintrack.git
git push -u origin main
```
### 
Push an existing repository from the command line

